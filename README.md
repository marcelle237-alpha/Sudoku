# Sudoku  
A browser-based Sudoku puzzle game built with HTML, CSS, and JavaScript.

## 🎯 Overview
Sudoku is a classic logic-based number placement game.  
This project implements a fully interactive Sudoku board where players can fill in numbers, validate moves, and solve puzzles directly in the browser.

It is designed to practice DOM manipulation, grid logic, and algorithmic thinking in JavaScript.

## 🚀 Features
- 9×9 interactive Sudoku grid  
- Input validation  
- Highlighting of rows, columns, and blocks  
- Error detection for invalid moves  
- Clean and responsive UI  
- Fully client-side (no backend required)

## 🧩 Technologies Used
- **HTML5** – structure  
- **CSS3** – layout & styling  
- **JavaScript (ES6)** – game logic & interactivity

## 🕹️ How to Play
1. Click a cell in the grid.  
2. Enter a number from 1 to 9.  
3. The game will highlight conflicts:
   - Same number in the row  
   - Same number in the column  
   - Same number in the 3×3 block  
4. Continue until the puzzle is solved.
