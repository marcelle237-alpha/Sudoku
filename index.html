How It Works

Fill every square with a number from 1 to 9. Each row, each column, and each 3x3 box must contain every number once.

Pick a level, select an empty square, then use the number pad or keyboard. Notes lets you pencil in possible answers, Check reviews your progress, and Hint fills one square.
